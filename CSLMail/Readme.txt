CSLMail Readme - 10th January 2006
==================================

The "Assemblies" folder contains the various assemblies for a variety of platforms - desktop and compact device. Reference the correct assembly for your use.

"HelpFile" folder contains the Help 2.0 based help file - compatible with VS 2005 help system. Execute the "help20register.bat" file before using the "CSLMail Documentation" shortcut for the first time.
